## 🎯 使用场景

### 场景 1: 新用户注册
1. 访问注册页面
2. 填写用户信息
3. 获取验证码
4. 输入验证码
5. 完成注册
6. 自动登录

### 场景 2: 已有用户登录
1. 访问登录页面
2. 输入用户名密码
3. 点击登录
4. 跳转到首页

### 场景 3: Token 持久化
1. 登录成功
2. 刷新页面
3. 自动保持登录状态

### 场景 4: Token 过期
1. API 返回 401
2. 自动清除 Token
3. 重定向到登录页
