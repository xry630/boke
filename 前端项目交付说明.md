## 📋 API 端点对接

前端已完美对接以下后端 API：

1. **POST /api/Auth** - 用户登录
2. **POST /api/Auth/register** - 用户注册  
3. **GET /api/Code** - 获取验证码

所有 API 请求自动携带 JWT Token（通过 Axios 拦截器）。
